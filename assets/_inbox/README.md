# Asset inbox

Drop ANY downloaded asset files here — HDRIs, PBR texture sets, GLBs,
zipped packs from Quaternius / KayKit / Kenney, individual files from
poly.pizza. Original filenames are fine; I'll inspect, rename, and move
them to the right manifest paths.

Subfolders are also fine. If you've extracted a Quaternius zip, just
drop the whole extracted folder in here and I'll find what I need.

This folder is gitignored — nothing here will be committed.
